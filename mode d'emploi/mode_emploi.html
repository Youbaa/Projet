A - Mode d'utilisation


Présentation de l'application

- Cette page constitue le mode d'utilisation de notre application ToolCrypt, l'outil développé dans le cadre du projet "Boite Noire" Master 1 de fin d'année spécialité "Sécurité des Systèmes Informatiques" à l'université Paris-Est Créteil (UPEC)


- Cet outil a été développé en Java et permet à un cryptanaliste d'introduire des fichiers crypté et à l'aide des différentes fonctionnalitées ( découpage par bloc, analyse par fréquence, substitution...) il pourra avoir des sorties exploitable qui vont l'aider à décrypter le fichier.



Présentation de l'interface graphique


L'interface graphique de notre outil contient un volet d'exploration qui contient Fichier et Aide, un accée à l'explorateur windows pour trouver le fichier texte sur lequel vous voulez travailler, un menu déroulent avec les différentes fonctionnalités, ainsi que deux anglets, le premier "Résultat" pour visualisez le liens du fichier sortant, et le deuxième "Historique" pour revoir les différentes opérations effectuer.

B - Fonctionnalités


Pour expliquer les différentes fonctionnalités nous avons utiliser un fichier texte crypyé qui se trouve ici

Découpage en bloc

-Séléctionner le fichier,
-choisir la fonctionnalité "découpage par bloc",
-excecuté l'application,
-choisir la taille de chaque bloc



Comme sortie on aura le résultat afficher sur la console et le lien d'un fichier texte qui contient ce résultat



Analyse de frequence par bloc

-Séléctionner le fichier,
-choisir la fonctionnalité "Analyse de fréquence par bloc",
-excecuté l'application,
-choisir la taille de chaque bloc
-Choisir la position



Comme sortie on aura le résultat afficher sur la console



Analyse de frequence par caractère

-Séléctionner le fichier,
-choisir la fonctionnalité "Analyse de fréquence par caractére",
-excecuté l'application,
-Visualier le resultat sur la console (il donne pour chaque caractére le nombre de répetition ainsi que sa fréquence dans le texte)
